The Python Software Foundation has presented a [Community Service Award](http://www.python.org/community/awards/psf-awards/) for the first quarter of 2011 to [Benjamin Peterson](http://www.benjamin-peterson.org/) for his work as Release Manager and the numerous contributions to the Python Core.  
The Python Software Foundation is pleased to recognize Benjamin's contributions to the community.
