Join [Tania Allard](https://www.trallard.dev/), PSF Board Member, and [Łukasz Langa](https://lukasz.langa.pl/), CPython Developer-in-Residence, for ‘[Python 3.13 and the Latest Trends: A Developer’s Guide to 2025](https://www.youtube.com/live/GPwYSf1t8Lw?si=USUFEcOEgRrhuCxM)’, a live stream event hosted by Paul Everitt from [JetBrains](https://www.jetbrains.com/). Thank to JetBrains for partnering with us on the Python Developers Survey and this event to highlight the current state of Python!  

The session will take place tomorrow, October 3, at 5:00 pm CEST (11:00 am EDT). Tania and Łukasz will be discussing the [exciting new features in Python 3.13](https://docs.python.org/3.13/whatsnew/3.13.html), plans for Python 3.15 and current Python trends gathered from the [2023 Annual Developers Survey](https://lp.jetbrains.com/python-developers-survey-2023/). Don't miss this chance to hear directly from the experts behind Python’s development!
